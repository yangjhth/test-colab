## üå≥ Tree Planting Action Checklist

### One-Time Setup

**Tool Inventory:**
- [ ] Shovel(s)
- [ ] Work gloves
- [ ] Watering can or hose
- [ ] Mulch bags
- [ ] Stakes (if needed)
- [ ] Ties for stakes
- [ ] Fertilizer
- [ ] Wheelbarrow (optional but helpful)

**Site Preparation:**
- [ ] Mark each planting location
- [ ] Measure spacing between spots
- [ ] Clear any debris from planting areas
- [ ] Have water source accessible

---

### Planting Session Checklist

**Before You Start:**
- [ ] Check weather (avoid extreme heat or rain)
- [ ] Wear comfortable clothes that can get dirty
- [ ] Apply sunscreen
- [ ] Fill water bottles
- [ ] Set timer for 2-hour focused session

**For Each Tree (Repeat 2-3 times per session):**
1. [ ] Dig hole 2-3x wider than root ball
2. [ ] Dig hole as deep as root ball
3. [ ] Loosen soil around hole
4. [ ] Remove tree from container gently
5. [ ] Loosen root ball if needed
6. [ ] Place tree in hole (check depth)
7. [ ] Backfill with soil halfway
8. [ ] Water thoroughly
9. [ ] Finish backfilling
10. [ ] Water again
11. [ ] Add mulch around base (not touching trunk)
12. [ ] Stake if needed
13. [ ] Take photo for records
14. [ ] Mark as completed on tree list

**After Session:**
- [ ] Clean and store tools
- [ ] Hydrate yourself!
- [ ] Mark progress on calendar
- [ ] Celebrate! (treat yourself)
- [ ] Schedule next planting session if more trees remain

---

### Ongoing Maintenance Checklist

**Weekly (First Month):**
- [ ] Water deeply 2-3 times per week
- [ ] Check for any issues (insects, damage)
- [ ] Take weekly progress photo

**Monthly (Ongoing):**
- [ ] Water as needed based on rainfall
- [ ] Check mulch levels
- [ ] Inspect for pests or disease
- [ ] Remove weeds around base
- [ ] Check stakes/ties (adjust if needed)

**Quarterly:**
- [ ] Fertilize per tree type requirements
- [ ] Prune if necessary
- [ ] Add mulch if depleted
- [ ] Take quarterly growth photo

### Phase 1: Exploration (First 6 Months)

**Week 1-2: Library Reconnaissance**
- [ ] Look up local library's 3D printer policy online
- [ ] Call library to ask about:
  - [ ] Reservation process
  - [ ] Cost (if any)
  - [ ] Training/orientation required?
  - [ ] Time limits per session
  - [ ] File format requirements
- [ ] Schedule first orientation/visit
- [ ] Create observation notebook (physical or digital)

**First Library Visit:**
- [ ] Attend orientation/training
- [ ] Ask questions about printer type and capabilities
- [ ] Observe a print being made
- [ ] Get sample files or recommended beginner files
- [ ] Take photos of the printer and setup
- [ ] Note printer brand/model for research
- [ ] Schedule first printing session

**First 3 Printing Sessions:**
- [ ] Session 1: Print a simple object (cube, keychain)
  - [ ] Document: time, settings, observations
  - [ ] Note challenges or questions
- [ ] Session 2: Print something slightly more complex
  - [ ] Try different settings or material
  - [ ] Compare to first print
- [ ] Session 3: Print own design or modified file
  - [ ] Document learning curve
  - [ ] Note what you'd like to improve

**Learning & Research (Parallel Track):**
- [ ] Watch 3 beginner YouTube tutorials on 3D printing
- [ ] Watch 2 videos on 3D modeling software
- [ ] Research local maker spaces or 3D printing groups
- [ ] Join online community (Reddit, forum, etc.)
- [ ] Download free 3D modeling software to explore
- [ ] Try modifying an existing STL file

**Month 3-6: Regular Practice**
- [ ] Set recurring calendar time: 2-4 hours/week
- [ ] Attempt 1 print per week (minimum)
- [ ] Try at least 3 different types of objects
- [ ] Learn one new 3D modeling technique per month
- [ ] Connect with one person in maker community
- [ ] Document all prints in learning log

**By End of Phase 1:**
- [ ] Completed at least 10 prints
- [ ] Comfortable with basic printing process
- [ ] Understand different printer types (FDM vs SLA)
- [ ] Can modify existing 3D models
- [ ] Have started creating simple original designs
- [ ] Made decision on future printer purchase direction
