## Quick Start Guide

### For Tree Planting Days:
1. Check weather forecast
2. Gather tools (shovel, gloves, mulch, stakes, water)
3. Set timer for focused 2-hour session
4. Plant 2-3 trees
5. Celebrate the win!

### For 3D Printing Learning Days:
1. Reserve library 3D printer time (if applicable)
2. Bring notebook for observations
3. Watch one tutorial before going
4. Try printing one simple object
5. Document learnings in THOUGHTS.md

### For Crabbing/Fishing Planning Days:
1. Research one aspect (location, timing, or logistics)
2. Add to checklist
3. Review with family if possible
4. Set tentative date for outing
